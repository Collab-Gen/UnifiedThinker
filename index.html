Abstract

尽管高保真图像合成取得了显著进步，但在逻辑密集型指令遵循方面仍面临挑战 [cite: 1, 349]。 我们提出了 Unified Thinker，这是一种任务无关的通用图像生成推理架构，设计为可插入不同生成器和工作流的统一规划核心 [cite: 4, 352]。 Unified Thinker 实现了“思考者（Thinker）”与“生成器（Generator）”的解耦，允许在不重新训练整个模型的情况下进行模块化推理升级 [cite: 5, 353]。 我们引入了两阶段训练范式：首先构建结构化规划接口，然后应用强化学习（RL）将策略锚定在像素级反馈中，优化视觉准确性 [cite: 6, 354]。

Visual Demonstrations

Figure 1: Unified Thinker 在图像编辑和文本生成图像任务中的视觉演示。它能够处理复杂的逻辑指令，如数独求解或基于常识的场景修改 [cite: 22, 442]。

Methodology: Think-then-Execute

我们的框架由两个核心组件组成：

  • Thinker: 一个独立的可训练多模态大语言模型（MLLM），将指令转换为层次化的、生成器友好的规划（包括意图总结、显式约束和子目标） [cite: 25, 445]。
  • Generator: 基于扩散模型的图像合成后端，根据 Thinker 的输出生成最终图像 [cite: 24, 444]。

Figure 2: 提出的两阶段训练框架：联合监督微调（Joint SFT）和执行引导的双相强化学习（Dual-Phase RL） [cite: 77, 329]。

Key Results

在多个基准测试（如 RISEBench, WiseBench）中，Unified Thinker 显著提升了指令遵循和约束满足能力 [cite: 32, 452]：

  • 推理增强：在 RISEBench 上，相比基础生成器，在时间、因果和空间推理维度均有大幅提升 [cite: 106, 115]。
  • 模块化迁移：该推理核心在不同生成器后端（如 Qwen2.5-VL-7B, Qwen3-VL-8B）上表现出良好的跨模型移植性 [cite: 119, 439]。
